* IO 

学习就两个步骤, 输入和输出. 输入源在哪其实不是很重要, 重要的是

- 文件分类
  - todo 前缀: 无知识转换的 todo 清单, 仅仅是去做某事, 但是做某事也不是那么容易的, 有的事情会耗费个几天. 例如搞某些材料
  - git 前缀: 代表具体的项目
  - 自学
    - xxoo : 知识体系, 即代表了 io 中的 iuput.
    - xxoo-internal: XXOO 的训练题, 实验过程等
